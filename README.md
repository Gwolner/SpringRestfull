# Consumo da API

## Servico 

### Inserir

Link: /Servico
Requisição: POST

Mandar 
```json
{
 "designacao":"Abrigo"
}
```

Receber
```json
Snipet
```

### Atualizar

Link: /Servico
Requisição: PUT

Mandar 
```json
{
 "id":"1",
 "designacao":"Abrigo"
}
```

Receber
```json
Snipet

```

### Ler

Link: /Servico/{id}
Requisição: GET

Mandar 
```json
Snipet
```

Receber
```json
{
 "id": 1,
 "designacao": "Abrigo"
}

```

### Deletar

Link: /Servico/{id}

Snipet
